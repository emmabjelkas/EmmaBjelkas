CASE STUDY
- STALLBACKEN

Introduction

Stallbacken is a somewhat known website in the equestrian world that gathers useful tools and information about competition, both for riders and organizers. The idea of the website is very useful and could make it easier for equestrians to find all the information they need if they're new to the competitive world. However, when visiting the website, I couldn't help but notice that the experience wasn't as fulfilling as i had hoped for a website people recommended to others. I saw this as an oppertunity to re-imagine the digital experience and the way that people engage with Stallbackens content.

Challenge

On first glance, the website really pops! Unfortunatly not in a good way. The starting screen has a strong color that doesn't feel inviting and it's straining on the eye. The font used is the classic Times New Roman, but it's hard to read mostly because of the sizing. Everything is written in the same size, even the titles in the menu.

Discovery

Opon doing my research, I discovered that the website is run by one single person, not a team of people. So here I was optimistic and joyful because the website is simply made by a person that has a burning intrest for what they're creating. I decided to start with one objective for the redesign; lessen the cognitive load and give it a "cleaner" look. Later on I would love to further expand on my work.

Strategy

1. Create a more clean look with less cognitive load and easier to read texts.

2. Elevate the overall experience to align with the reputation of the website.

Solutions

1. Chose a neutral color palette.

Since the website is made to give useful information about competitions, it needs to come across as trust worthy. Therefore I would pick blue as the main color for the website. The color blue is used for multiple brands in the equestrian industry, such as the very brand that writes the rules for competitions - Svenska Ridsportsförbundet.

Below follows an example of a blue color palette perfect for the site.

2. Create clearer hierarchy and more white space.

When visiting the website, the user should quickly understand what the website is about and what its purpose is. The user should also easily find information about their discipline, such as dressage or showjumping. The menu containing links to each discipline are placed on top of the screen so a frequent user quickly can access them. By adding a clearly visable Headline or Hero Image, it takes no longer than a few seconds for the user to recieve this information. I make sure to add this below the context of the menu.
The website has very little use of white space. Titles, links, pictures and information are simply put in one single group. It also doesn't use a clear hierarchy for all the elements on the website. The one thing they use is dividing sections with boxes and lines to make it easier for the user to see what elements go together. By simply using different sizing of the different titles and paragraphs, there is no need to divide them with lines.
By removing most of the overwhelming information on the starting screen and putting them in categories in the menu, the cognitive load is redused. An exampel is the information about pricing in the shop, I removed this from the starting screen and put it on the "Shop" page.

3. Accessibility.

Using a Sans-Serif font, such as Inter and Instrument Sans, would make the text easier to read and Therefore more accessibile for people with visual impairment.
As mentioned earlier, different size fonts are good to use, but also different thickness. A bolder Headline to make it stand out and a thinner font for paragraphs to not overwhelm the user with information.

Other than changing the font, I would use a new color palette that passes WCAG AA standards for color contrast. As of right now, there are texts that fail these standards. On the bottom of the screen they use a text in the color #38AA76 against the background color #FFFFCC, which has a ratio of 2.84:1. In order to pass the WCAG AA standards for normal text, it needs a ratio of 4.5:1.

The design

Implementing the changes mentioned above, I put together a mockup that you can see below. The biggest difference is the clearer hierarchy, a clearly visable headline and bigger use of whitespace.

Impact

By transitioning from a cluttered layout to a structured, WCAG-compliant design, the redesign achieves the core goals:
The new color palette moves from a failing $2.84:1$ ratio to a $4.5:1$ (AA) standard, ensuring that the site is usable for equestrians with visual impairments. By moving secondary information (like shop pricing) to dedicated sub-pages and utilizing whitespace, the "time-to-task" for finding competition rules is significantly reduced.
